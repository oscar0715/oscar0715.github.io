Maven简介

Maven 主要服务于Java平台的项目构建，依赖管理和项目信息管理。

##构建

构建（build）除了编写源代码，我们每天花很多时间在编译，运行单元测试，生成文档，打包和部署等工
作上，这就是构建。

Maven 定义了了完整的构建生命周期，能够自动化构建，实现构建任务，于是程序猿不必花时间在定义构建过程上。
