Network Flows
网络流问题

Flow Network （流网络）

A flow network is a connected, directed graph G = (V , E ).
一个网络流是一个连接有向图

  1. 每条边有一个capacity cec_ec​e​​，是一个非负整数
  2. A single source s∈V. 起点
  3. A single sink t∈V. 终点
  4. No edge enters the source and no edge leaves the sink
    起点无入边，终点无出边

假设

  1. Capacities are integers. 每条边的容量为整数
  2. Every node has one edge adjacent to it. 每个顶点都有相邻的边
  3. No edge enters the source and no edge leaves the sink. 起点无入边，终点无出边

Flow（流）

把从sss 到 ttt的流看作是一个function，fff ，这个function给每条边的指定了一个整数的流量。
f(e)f(e)f(e)就是经过eee这条边的流量。

Constraints on fff :

  1. 流量限制
    0 ≤ f (e) ≤ c_e for each edge eee.
  2. 平衡限制
    对于除了sss 和 ttt 以外的顶点 vvv，有：
$\sum_{e \text{ into } v} f(e) = \sum_{e \text{ leaving } v} f(e) $

Notation:

  1. f的值：
    f(e)=∑e out of sf(e)f(e) = \sum_{e \text{ out of } s} f(e)f(e)=∑​e out of s​​f(e)
    即从顶点sss送出的流量
  2. fin(v)=∑e into vf(e)f^{in}(v) = \sum_{e \text{ into } v} f(e)f​in​​(v)=∑​e into v​​f(e)
  3. fout(v)=∑e leaving vf(e)f^{out}(v) = \sum_{e \text{ leaving } v} f(e)f​out​​(v)=∑​e leaving v​​f(e)
  4. 于是平衡限制可以写成fin(v)=fout(v)f^{in}(v) = f^{out}(v)f​in​​(v)=f​out​​(v)

Problem（最大流问题）

Definition (Value)：
The value v(f)v(f)v(f) of a flow fff is fout(s)f^{out}(s)f​out​​(s).

Problem (Maximum Flow)：
Given a flow network GGG, find a flow fff of maximum possible value.
给定一个图GGG，求fff的最大值，也就是fout(s)f^{out}(s)f​out​​(s)的最大值。起点最多能送出多少流量。

就是每次根据原图 GGG 的 fff ，构成新的 GfG_fG​f​​ ，再由这个新的 GfG_fG​f​​ 去更新原图 GGG 的 fff

Cuts Analysis

Cuts and Cut Capacity

Definition
The capacity(A,B) of an s-t cut (A,B) is the sum of the capacities of edges leaving A.

  1. 将图 GGG 中的点分为两部分，AAA 和 BBB，s∈As \in As∈A, t∈Bt \in Bt∈B
  2. 这就是一个 s−ts-ts−t cutcutcut：(A,B)(A,B)(A,B)
  3. cut(A,B)cut(A,B)cut(A,B) 的最大容量为 c(A,B)=∑e out of Acec(A,B) = \sum_{e \text{ out of } A } c_ec(A,B)=∑​e out of A​​c​e​​，即从 AAA 流向 BBB 的流量的总和

Cut Theorem

一条定理 Theorem
Let fff be an s−ts-ts−t flow and (A,B)(A,B)(A,B) be an s−ts-ts−t cutcutcut.
Then v(f) = f^{out}(A) − f^{in}(A).

证明：

  1. 对于原点来说
    fin(s)=0f^{in}(s) = 0f​in​​(s)=0，所以v(f)=fout(s)−fin(s)v(f) = f^{out}(s) - f^{in}(s)v(f)=f​out​​(s)−f​in​​(s)
  2. 对于 AAA 中的其他的点 vvv 来说
    这些点都是 GGG 中，所以根据 Network flow 的定义有，fout(v)−fin(v)=0f^{out}(v) - f^{in}(v) = 0f​out​​(v)−f​in​​(v)=0
  3. 所以
    我们得到 v(f)=∑v∈A(fout(v)−fin(v))v(f) = \sum_{v \in A} (f^{out}(v) - f^{in}(v))v(f)=∑​v∈A​​(f​out​​(v)−f​in​​(v))
    其实只有 v=sv=sv=s 的时候差值才非零

对于一条的边 eee 来说：

  1. 如果 eee 的两端都在 AAA 中，那么 eee 在上述的求和公式中，一次起到加的作用，一次起到减的作用，抵消掉了。
  2. 如果 eee 只有头在 AAA 中，那么就代表流量流入 AAA ，那么起到加的作用
  3. 反之 eee 只有尾在 AAA 中，则代表流量流出 AAA ，起到减的作用
  4. 如果两端都不在 AAA ，那么对求和没影响

我们可以改写上述的式子

$v(f) = \sum_{v \in A} (f^{out}(v) - f^{in}(v)) = \sum_{e \text{ out of } A} - \sum_{e \text{ into } A} = f^{out}(A) - f^{in}(A)$

对于BBB来说则刚好相反
v(f)=fin(B)−fout(B)v(f) = f^{in}(B) - f^{out}(B)v(f)=f​in​​(B)−f​out​​(B)

另一条定理 Theorem
Let fff be any s−ts-ts−t flow and (A,B)(A,B)(A,B) be any s−ts-ts−t cutcutcut.
Then v(f) ≤ capacity(A, B).

证明：

$ \begin{align*} v(f ) & = f^{out}(A) - f^{in}(A) \\ & \leqslant f^{out}(A) \\ & = \sum_{e \text{ leaving } A} f(e) \\ & \leqslant \sum_{e \text{ leaving } A} c_e \\ & = capacity(A,B) \end{align*} $

这条定理很有用，虽然它只是个不等式，但是不等式右边独立于任何一条具体的流 fff。
也就是说明了，任何的流 fff 都的上限都不会超过任何一个 cutcutcut 的capacity

Max Flow = MinCut

  1. 如果在residual Graph GfG_fG​f​​ 中无法找到一条从 sss 到 ttt 的路径，那么此时得到的fff，就是流量最大的 fff ，记为 f∗f^*f​∗​​.
  2. 此时，必定有一道$cut(A^*, B^*)$，使得$v(f) = capacity(A^*, B^*)$
  3. 而且此时，每条从 AAA 到 BBB 的边都是饱和的，否则在 GfG_fG​f​​ 就有一条正向边，那么还存在一条 Augmenting Path
  4. 每条从 BBB 到 AAA 的边都是的流量都是 0，否则在 GfG_fG​f​​，就有一条从 BBB 到 AAA 的反向边，也就是从 AAA 到 BBB 的边，那么又存在一条Augmenting Path了
$$ \begin{align*} v(f ) & = f^{out}(A^*) - f^{in}(A^*) \\ & = \sum_{e \text{ out of } A^*} f(e) - \sum_{e \text{ into } A^*} f(e)\\ & = \sum_{e \text{ leaving } A^*} c_e - 0 \\ & = capacity(A^*,B^*) \end{align*} $$

所以我们可以看见，在残差图中的正向边和反向边是多么的重要的。
我们可以得到以下结论

  1. Ford-Fulkerson Algorithm得到的流 f∗f^*f​∗​​ 是最大流
  2. 如果已知最大流的值，那么我们可以算出 s−ts-ts−t cutcutcut 的最小容量，时间复杂度为 O(m)O(m)O(m)
    只需在最后得到的 residual Graph 中从 sss 开始用BFS或者DFS搜索，能达到的点属于$A^*$，剩下的点属于$B^*$
    需要注意的是，可能有很多个这样的cut，上述方法只能找到其中的一条

总结一下

求一个 Min-capacity Cut的流程

  1. 找到最大流 f∗f^*f​∗​​
  2. 根据这个最大流构建残差图 Gf∗G_{f^{*}}G​f​∗​​​​
  3. 从起点 sss 出发做BFS，能够搜索到的点都属于 A∗A^{*}A​∗​​
  4. 剩下的点属于 B∗{B^{*}}B​∗​​
  5. ${A^{*}, B^{*}}$就是我们要找的Min-capacity Cut
