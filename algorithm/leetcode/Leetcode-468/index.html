Validate IP Address

Problem

给定一个字符串
判断是 IPv4 还是 IPv6，还是 “Neither”

Solution

头尾不能是’.’ 或者 ‘:’

IPv4:

  1. 必须四段
  2. 必须是小于等于255的正整数，或者一个0
  3. 不能有 leading 0

IPv6:

  1. 必须有8段
  2. 长度必须小于等于4，不能为空
  3. 要么是数字，要么是 ‘a-f’的字母，大小写都可以
